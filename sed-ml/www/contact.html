Contact us

SED-ML is a community effort, involving software developers of various simulation tools and model representation formats. Our effort values all opinion and contributions. If you have a question, want to discuss the evolution of SED-ML, want to voice an opinion, write and subscribe to sed-ml-discuss on googlegroups.

The SED-ML specification is written by an editorial board. If you want to directly contact the editors, please send a mail to sed-ml-editors AT googlegroups.com.

To report a bug or record an issue with the current specification, please submit an item to the L1V2 tracker.

To record a feature request with future specifications, please submit an item to the feature request tracker.
