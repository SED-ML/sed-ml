\label{sec:listOfChanges}
%
\sedfig[width=0.7\textwidth]{listOfChanges}{The SED-ML listOfChanges container}{fig:listOfChanges}
%
The \concept{listOfChanges} container contains the defined changes to be applied to a particular \hyperref[class:model]{model} (see \fig{listOfChanges}). \concept{listOfChanges} always occurs within a \element{model} element. It is an optional subelement of the \element{model} element.

Example \ref{lst:listOfChanges} shows the definition of a \element{listOfChanges} container inside a \element{model} element.
\begin{myXmlLst}{The SED-ML \element{listOfChanges} element, defining a change on a model}{lst:listOfChanges}
<model id="m0001" [..]>
 <listOfChanges>
  [CHANGE DEFINITION]
 </listOfChanges>
</model>
\end{myXmlLst}
%
In the example a change is defined on the model \element{m0001}. To encode the change, a \element{listOfChanges} is created that contains all changes in single \element{change} elements, as explained in the definition of the \hyperref[class:change]{Change} class.
