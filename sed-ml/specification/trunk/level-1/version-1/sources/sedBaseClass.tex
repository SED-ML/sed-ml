% SED-Base class
\label{class:sedBase}
\concept{SEDBase} is the base class  of SED-ML \LoneVone. All other classes are derived from it. It provides means to attach additional information on all other classes  (\fig{sedBase}). That information can be specified in form of human readable \hyperref[class:notes]{Notes} or custom \hyperref[class:annotation]{Annotation} classes.

\subsubsection{\code{metaid} Attribute}
\label{sec:metaID}
The main purpose of the \element{metaid} attribute is to attach instances of the \hyperref[class:annotation]{Annotation} class to SED-ML elements. 
The \code{metaID} attribute is globally unique throughout the whole SED-ML document. 

\code{metaID} is of type XML ID.
