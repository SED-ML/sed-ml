Contact points

SED-ML is a community effort, involving software developers of various simulation tools and model representation formats. The effort values all opinion and contributions. If you want to discuss the evolution of SED-ML, have a question about the current specification, of voice an opinion, please subscribe to sed-ml-discuss on SourceForge.

The SED-ML specification is written by an editorial board. If you want to directly contact the editors, please send a mail to sed-ml-editors AT sf.lists.net.

To report a bug or record an issue with the current specification, please submit an iterm to the L1 V1 tracker.

To record a feature request with future specifications, please submit an iterm to the feature request tracker.
