\label{sec:xmlns}
The \concept{xmlns} attribute is used in several places of the SED-ML document.
First of all, it declares the namespace for the SED-ML document. The pre-defined standard namespace is \url{http://www.biomodels.net/sed-ml}. 

In addition, SED-ML makes use of the \concept{MathML} namespace \url{http://www.w3.org/1998/Math/MathML} to enable encoding mathematical expressions in MathML 2.0. SED-ML uses a subset of MathML as described in section \ref{sec:mathML} on page \pageref{sec:mathML}.

For the \hyperref[class:notes]{Notes} class, the standard value of \code{xmlns} is \url{http://www.w3.org/1999/xhtml}.
